Team name : Rogue Minds
A Modern Interactive Front-End Web Experience
Overview
Rogue Minds is a handcrafted front-end web project developed using HTML5, CSS3, JavaScript, GSAP, and Lenis.
The project focuses on delivering a visually immersive and smooth interactive experience through advanced animations and fluid scrolling effects. It demonstrates modern web development practices by combining structured markup, responsive styling, and motion-based user interaction.
This project was built entirely through manual coding without AI-generated development, emphasizing originality and technical implementation.
